## Assessment

Create a **20-minute workshop** in your group using LiaScript and GitHub:

**Requirements:**

- [ ] Use videos, media (pictures, etc.), and interactive options
- [ ] Address a professional development learning objective
- [ ] Define precise learning goals
- [ ] Gather relevant material with examples and scenarios
- [ ] Design course following **Bloom's Taxonomy**
- [ ] Include didactical and methodological elements
- [ ] Implement knowledge assessment and success measurements

**Your Task:**

1. Define learning goals precisely
2. Gather relevant materials
3. Choose relevant examples and scenarios
4. Enable users to develop competencies
5. Design a well-thought-through course structure
6. Include internal assessments
